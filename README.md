# Flip 7 Calculator

A score calculator for the Flip 7 card game.

## Features

- Animated splash screen with 7 cards fanning out
- Add and delete players
- Enter each round's cards to calculate scores, including the Flip 7 bonus
- New game (keeps players, resets scores) and full reset
- Works entirely offline, scores persist via localStorage

## Getting Started

### 1. Install dependencies

```bash
npm install
```

### 2. Start development server

```bash
npm run dev
```

### 3. Build for production

```bash
npm run build
```

## Deploying

### Vercel

1. Push to GitHub
2. Import to Vercel
3. Deploy — no environment variables needed

## Tech Stack

- React + TypeScript + Vite
- Tailwind CSS (pastel theme)
- Framer Motion (animations)
- lucide-react (icons)
